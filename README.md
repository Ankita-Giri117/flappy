# 🐦 Flappy Bird Clone

A simple browser-based Flappy Bird game built using **HTML, CSS, and JavaScript**.

Play the game here 👉""

---

## 🎮 Features

* Smooth jump physics
* Random pipe generation
* Score tracking
* Sound effects & background music
* Game over screen with restart button
* Keyboard and mouse controls
* Fully playable in browser (no installation)

---

## 🕹️ Controls

* **Spacebar** → Jump
* **Mouse Click** → Jump

Avoid the pipes and try to score as high as possible!

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations & Layout)
* JavaScript (Game Logic & Collision Detection)

## 🚀 How to Run Locally

1. Download or clone the repository

```
git clone https://github.com/Ankita-Giri117/flappy-bird.git
```

2. Open `index.html` in your browser
